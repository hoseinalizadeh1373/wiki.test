# IncomingCall

هنگامی‌ كه تماسی به سیموتل وارد شود (تماس ورودی)، رویداد IncomingCall ایجاد می‌گردد و مشخص می‌نماید كه تماس از چه طریقی‌ (entry_point) وارد شده است.
